# Dyslexia-Friendly Adaptive Reading System

This project implements a web-based assistive reading system designed to support dyslexic learners in higher education. The system transforms standard academic documents (PDF/TXT) into accessible, dyslexia-friendly formats using text transformation and multisensory assistive technologies.

## Key Features
- Dyslexia-friendly reading interface
- Selective syllable splitting for difficult words
- Difficult word highlighting
- Adjustable font size, spacing, and themes
- Focus mode for sentence-by-sentence reading
- Text-to-speech narration
- Accessible PDF export
- Upload support for PDF and TXT documents


## Technology Stack
- Python
- Streamlit
- PyPDF2
- Pyphen
- gTTS / pyttsx3
- ReportLab

## Usage

1. Upload a PDF or TXT document
2. Customize reading settings
3. Enable focus mode for guided reading
4. Use text-to-speech narration
5. Export the accessible PDF if needed

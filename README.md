# Dyslexia-Friendly Adaptive Reading System

This project implements a web-based assistive reading system designed to support dyslexic learners in higher education. The system transforms standard academic documents (PDF/TXT) into accessible, dyslexia-friendly formats using text transformation and multisensory assistive technologies.

## Key Features
- PDF and text document ingestion
- Customizable visual adaptation (font, spacing, themes)
- Syllable-based decoding using linguistic rules
- Synchronized text-to-speech with real-time highlighting
- One-line focus mode to reduce visual distraction
- Stateless, privacy-first document processing

## Technology Stack
- Python
- Streamlit
- PyPDF2
- Pyphen
- gTTS / pyttsx3
